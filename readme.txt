Este simulador es para reservar un vuelo en avión.

Se puede Reservar y Consultar

*Reservar:

1. Se ingresan los datos del vuelo
2. Se ingresan los datos del pasajero
    a. Valida DNI
    b. Valida numero de telefono
    c. Valida mail
3. Se ingresan los datos de pago
    a. Valida número de tarjeta (16 digitos)
    b. Valida codigo de seguridad (3 digitos)
4. Muestra un resumen del vuelo antes de confirmar la compra
5. Muestra la reserva exitosa y el numero de reserva para utilizarlo en el consultar.
